# Meta-Regression method for objects of class 'runMetaAnalysis'

Prints information that models have to be extracted from
`runMetaAnalysis` results objects to run meta-regression.s

## Usage

``` r
# S3 method for class 'runMetaAnalysis'
metaRegression(x, formula = NULL, ...)
```

## Arguments

- x:

  An object of class `runMetaAnalysis`.

- formula:

  not used.

- ...:

  Additional arguments.
