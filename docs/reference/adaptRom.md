# Adapt model results from log ROM to ROM (exp and rename g -\> rom)

Adapt model results from log ROM to ROM (exp and rename g -\> rom)

## Usage

``` r
adaptRom(M, round.digits = 2)
```

## Arguments

- M:

  List with `m` and `res`; `res` has `g`, `g.ci`, `prediction.ci`.

- round.digits:

  Rounding digits for displayed values.
