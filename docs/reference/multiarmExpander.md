# Expander function for multiarm trials

Expands multiarm trial.

## Usage

``` r
multiarmExpander(study, condition.specification, group.indicator, group.names)
```

## Arguments

- study:

  data of one study, for one unique assessment point.

- condition.specification:

  trial condition specification.

- group.indicator:

  group indicator (IG or CG).

- group.names:

  group names (list).
