# Internal helper function to generate forest plots with EB estimates

Internal helper function to generate forest plots with EB estimates

## Usage

``` r
forestBlupPlotter(
  dat,
  res,
  sort,
  col.line,
  col.polygon,
  hetstat,
  leftlab,
  rightlab,
  summarylab,
  M.3l,
  threeLevel,
  eb.labels
)
```

## Details

Parts of the code in this function is based on van Aert et al. (2021;
Supplement).

## References

van Aert, R. C., Schmid, C. H., Svensson, D., & Jackson, D. (2021).
Study specific prediction intervals for random-effects meta-analysis: A
tutorial: Prediction intervals in meta-analysis. *Research Synthesis
Methods, 12*(4), 429-447.
