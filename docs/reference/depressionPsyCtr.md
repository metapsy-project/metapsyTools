# The 'depressionPsyCtr' dataset

An example data set containing a subset of the 2021 depression trials
database. The data set has been formatted following the [Metapsy data
standard](https://docs.metapsy.org/data-preparation/format/).

## Usage

``` r
data("depressionPsyCtr")
```

## Format

A `data.frame` with 102 rows and 46 variables.

## Details

The dataset can be used by the `runMetaAnalysis` function without any
additional preparation required.
