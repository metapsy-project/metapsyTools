# The 'psyCtrSubset' dataset (wide format)

An example dataset containing a subset of the 2021 depression trials
database. In constrast to, e.g., the `psyCtrSubset` dataset,
`psyCtrSubsetWide` is structured in the "wide(r)" format. The dataset
also contains columns with study characteristics that are important for
effect size calculation and more.

## Usage

``` r
data("psyCtrSubsetWide")
```

## Format

A `data.frame` with 175 rows and 64 variables.

## Details

The dataset has been included to showcase the correct formatting
necessary to use `metapsyTools`.
