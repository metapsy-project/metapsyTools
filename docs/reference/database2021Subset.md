# The 'database2021Subset' dataset

An example dataset containing a subset of the 2021 depression trials
database. Its format equals the format of data in the "Metapsy"
database.

## Usage

``` r
data("database2021Subset")
```

## Format

A `data.frame` with 381 rows and 63 variables.

## Details

The dataset has been included to showcase the correct formatting
necessary to use `metapsyTools`.
