# Calculate Hedges' g using Mean and Standard Deviation.

Calculate Hedges' g using Mean and Standard Deviation. Only meant to be
used as part of [`calculateEffectSizes`](calculateEffectSizes.md).

## Usage

``` r
meanSD(x, ...)
```

## Arguments

- x:

  data

- ...:

  Effect size data. Must be `Post_M_trt1`, `Post_M_trt2`,
  `Post_SD_trt1`, `Post_SD_trt2`, `Post_N_trt1`, `Post_N_trt2` (all
  `numeric`).
