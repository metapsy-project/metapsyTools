# Get or set NNT control event rate

Generic getter and setter for the NNT control event rate (`nntCer`)
component of a `runMetaAnalysis` object. See
[`Replacement functions`](Replacement-functions.md).

## Usage

``` r
nnt(x, ...)

nnt(x) <- value
```

## Arguments

- x:

  Object of class `runMetaAnalysis`.

- ...:

  Optional arguments (ignored).

- value:

  New value for `nntCer`.

## Value

For `nnt(x)`, the current `nntCer` value. For `nnt(x) <- value`, the
updated object (invisibly).
