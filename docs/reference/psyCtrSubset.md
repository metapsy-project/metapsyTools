# The 'psyCtrSubset' dataset

An example dataset containing a subset of the 2021 depression trials
database. Its format equals the format of data in the "metapsy"
database, which is similar to the long format in *R*. The dataset also
contains columns with study characteristics that are important for
effect size calculation and more.

## Usage

``` r
data("psyCtrSubset")
```

## Format

A `data.frame` with 503 rows and 60 variables.

## Details

The dataset has been included to showcase the correct formatting
necessary to use `metapsyTools`.
