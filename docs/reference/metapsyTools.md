# metapsyTools: Several Helper Functions for `Metapsy` Databases

`metapsyTools` is a companion R package for the *Metapsy* database. It
aims to facilitate the calculation of meta-analytic effect sizes for the
included trials. For this purpose, data can be checked and converted
into the required format to calculate the effect sizes in R. It is then
possible to run a meta-analysis using the output effect sizes.
