# Function to generate forest plots with EB estimates

Function to generate forest plots with EB estimates

## Usage

``` r
forestBlup(
  model,
  which = NULL,
  col.line = "#a7a9ac",
  col.polygon = "#6b58a6",
  leftlab = "Study",
  rightlab = "g [95% CI]",
  summarylab = "Total (95% CI)",
  sort = TRUE,
  hetstat = TRUE,
  eb.labels = TRUE
)
```
