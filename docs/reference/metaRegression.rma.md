# Meta-Regression method for objects of class 'runMetaAnalysis'

Serves as a wrapper for `update.rma`.

## Usage

``` r
# S3 method for class 'rma'
metaRegression(x, formula = NULL, ...)
```

## Arguments

- x:

  A model extracted from an object of class `runMetaAnalysis`.

- formula:

  A `formula` object describing the predictor(s) to be added to the
  model. Default is `NULL`.

- ...:

  Additional arguments.
