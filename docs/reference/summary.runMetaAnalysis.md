# Show details of 'runMetaAnalysis' class objects

S3 method showing analysis settings for objects of class
`runMetaAnalysis`.

## Usage

``` r
# S3 method for class 'runMetaAnalysis'
summary(object, forest = TRUE, ...)
```

## Arguments

- object:

  An object of class `runMetaAnalysis`.

- forest:

  `logical`. Should a summary forest plot be returned? `TRUE` by
  default.

- ...:

  Additional arguments passed to
  [`meta::forest()`](https://rdrr.io/pkg/meta/man/forest.meta.html).
